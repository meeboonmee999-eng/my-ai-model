# 🚀 My Open Source AI Model
โมเดล AI ตัวแรกของฉันที่สร้างขึ้นบนมือถือ!

## 📌 เกี่ยวกับโปรเจกต์
โมเดลนี้ถูกสร้างขึ้นเพื่อ [บอกเป้าหมายของคุณ เช่น: แยกแยะข้อความภาษาไทย / วิเคราะห์ข้อมูลอุปกรณ์ช่าง] 
โดยเน้นความเบาและรันได้ง่าย

## 🛠 เครื่องมือที่ใช้
* **Language:** Python
* **Libraries:** Scikit-learn, Pandas, Joblib
* **Platform:** Developed on Android via Termux & GitHub

## 🚀 วิธีใช้งาน (Quick Start)
1. Clone โปรเจกต์นี้:
   `git clone https://github.com/meeboonmee999-eng/Meem...`
2. ติดตั้ง Library:
   `pip install -r requirements.txt`
3. รันโมเดล:
   `python predict.py`
